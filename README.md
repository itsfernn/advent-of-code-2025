# 🎄 Advent of Code 2025
Here are my entries for [Advent of Code 2025](https://adventofcode.com/2025/about) coding challange. 
This year, I’m writing all of my solutions in C, partly to push myself out of my comfort zone, and partly to get more practice working in a low-level, strongly typed language again.

To run a soltuion use the included `Makefile`
```bash
make
./puzzle[xx]/main
```
